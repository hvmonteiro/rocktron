CHAMELEON ON-LINE PRESETS


This chonline.syx file is a BULK DUMP of the new collection
of presets that are shipping with the new Chameleon On-Line
Preamp.  In order to install these new presets into your
original Chameleon, select the MIDI DUMP/LOAD function, then
Parameter Select BULK DUMP/LOAD.  Send the file from dumpster
like normal and you will see all the presets being loaded one
after another untill the display reads "USER DATA LOADED".

This file also contains the chonline.txt file which decribes
each presets title, config and parameter values in readable
form, which you may use to program your Chameleon manually.
