ORIGINAL CHAMELEON PRESETS


This chorigin.syx file is a BULK DUMP of the collection of 
factory presets that are shipped with the original Chameleon
Preamp.  In order to install these presets into your
Chameleon On-Line, select the MIDI DUMP/LOAD function, then
Parameter Select BULK DUMP/LOAD.  Send the file from dumpster
like normal and you will see all the presets being loaded one
after another untill the display reads "USER DATA LOADED".

This file also contains the chorigin.txt file which decribes
each presets title, config and parameter values in readable
form, which you may use to program your Chameleon manually.
